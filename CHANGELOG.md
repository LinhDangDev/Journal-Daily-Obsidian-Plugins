# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-02-26

### Added
- Journal entry creation with date-based folder structure (`Journal/YYYY/MM/date.md`)
- 5 built-in template presets (Full Journal, Minimal, Gratitude, Bullet, Weekly Review)
- Custom template creation and management with editor modal
- Template picker modal for choosing templates on entry creation
- Mood tracker with 5 levels (Great, Good, Okay, Sad, Awful)
- Calendar sidebar view with mood dots and monthly stats
- Journal Navigator with search, stats dashboard, and entry previews
- Writing streak tracker (current streak + longest streak)
- 50+ daily inspirational quotes via `{{quote}}` variable
- Status bar integration (journal status + streak counter)
- Multiple date format support (5 formats)
- Configurable settings (folder, features, behavior)
- Open journal on Obsidian startup option
- Ribbon icon for quick journal creation
